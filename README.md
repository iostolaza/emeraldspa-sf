# Emerald Spa SF Frontend

A compliant, static HTML/CSS front-end for Emerald Spa, located in San Francisco. 
Designed strictly as a Health & Wellness Spa platform, integrating with TidyCal for scheduling.

## Architecture
- Plain HTML5 / CSS3 (No heavy frameworks required)
- Vanilla JS for minimal interactions (Accordions)
- TidyCal integration for bookings

## Pages
- `index.html` (Landing)
- `about.html` (Story & Privacy Policy)
- `services.html` (Compliant Dermal & Body Practices)
- `packages.html` (Service bundles)
- `contact.html` (Location, Hours, FAQ)
